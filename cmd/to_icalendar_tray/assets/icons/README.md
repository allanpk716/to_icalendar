# Tray Icons

This directory contains system tray icons for the to_icalendar application.

## Required Icons

- `tray-16.png` - 16x16 pixels
- `tray-32.png` - 32x32 pixels
- `tray-48.png` - 48x48 pixels

## Icon Requirements

- Format: PNG with transparency support
- Style: Simple, recognizable calendar/task icon
- Color: Should work well on both light and dark backgrounds

These files are placeholder files that should be replaced with actual icons during development.
